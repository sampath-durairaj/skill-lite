Python Cheat Sheet

A condensed reference for core Python syntax, NumPy, Pandas, and Matplotlib — everything on one page.

Python core syntax

# Types
x = 1; x = 1.0; x = "s"; x = True; x = [1,2]; x = (1,2); x = {"k":"v"}; x = {1,2}; x = None

# f-strings
f"{name} is {age}"        f"{pi:.2f}"        f"{n:>5}"

# Control flow
if a > b: ...
elif a == b: ...
else: ...
for x in iterable: ...
while cond: ...
for i, x in enumerate(items): ...
for a, b in zip(list1, list2): ...

# Functions
def f(a, b=1, *args, **kwargs): return a + b
lambda x: x * 2

# Comprehensions
[x*x for x in range(10)]
{x: x*x for x in range(5)}
{x for x in range(10) if x % 2 == 0}

# Exceptions
try: ...
except ValueError as e: ...
finally: ...

# Classes
class C(Base):
    def __init__(self, x): self.x = x
    def method(self): return self.x

Common built-ins

FunctionPurpose

Matplotlib

import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(8,4))

ax.plot(x, y, label="y", color="#3776ab", linewidth=2, linestyle="--")
ax.scatter(x, y)
ax.bar(labels, values)
ax.hist(data, bins=20)
ax.boxplot(data)

ax.legend()   ax.grid(True, alpha=0.3)
ax.set_xlim(0, 10)   ax.set_ylim(-1, 1)

fig, axes = plt.subplots(1, 2, figsize=(10,4))   # multiple subplots
fig.tight_layout()

Environment & packages

python -m venv .venv
.venv\Scripts\activate          # Windows
source .venv/bin/activate       # macOS / Linux

pip install numpy pandas matplotlib
pip freeze > requirements.txt
pip install -r requirements.txt
Import aliases: import numpy as np, import pandas as pd, and import matplotlib.pyplot as plt are near-universal conventions — use them so your code reads consistently with the rest of the ecosystem.
