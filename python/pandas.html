Pandas

Pandas is Python's data-analysis library, built on top of NumPy. Its two core structures — Series (a labeled 1-D array) and DataFrame (a labeled 2-D table) — make loading, cleaning, filtering, and summarizing real-world tabular data fast and expressive.

pip install pandas

Series and DataFrame

import pandas as pd

s = pd.Series([10, 20, 30], index=["a", "b", "c"])
s["b"]                # 20 — label-based access

df = pd.DataFrame({
    "name":   ["Ada", "Grace", "Alan"],
    "age":    [36, 85, 41],
    "city":   ["London", "New York", "London"],
})
df.head()               # first 5 rows
df.shape                 # (3, 3) — rows, columns
df.columns                # Index(['name', 'age', 'city'])
df.dtypes                  # dtype of every column
df.info()                   # column types + non-null counts
df.describe()                 # count, mean, std, min, max for numeric cols

Selecting and indexing

df["name"]                 # single column → Series
df[["name", "age"]]          # multiple columns → DataFrame

df.loc[0]                     # row by label
df.loc[0, "name"]              # cell by label — 'Ada'
df.iloc[0]                      # row by position
df.iloc[0:2, 1:3]                 # rows 0-1, columns 1-2 by position

df.loc[df["age"] > 40]           # boolean filter
df[(df["age"] > 30) & (df["city"] == "London")]   # combine with & / |
df.query("age > 30 and city == 'London'")           # same, as a string expr

Adding, modifying, dropping columns

df["is_senior"] = df["age"] > 60
df["age_next_year"] = df["age"] + 1

df = df.drop(columns=["city"])
df = df.rename(columns={"name": "full_name"})
df = df.drop_duplicates()
df = df.sort_values("age", ascending=False)

Handling missing data

df.isna()                 # True/False mask for missing values
df.isna().sum()             # count of missing values per column
df.dropna()                   # drop rows with any missing value
df.dropna(subset=["age"])       # drop rows missing a specific column
df.fillna(0)                     # replace missing values with 0
df["age"].fillna(df["age"].mean(), inplace=True)   # fill with the column mean

Grouping and aggregation

df.groupby("city")["age"].mean()          # average age per city
df.groupby("city").agg({
    "age": ["mean", "max"],
    "name": "count",
})
df.groupby("city").size()                   # row count per group
df.pivot_table(values="age", index="city", aggfunc="mean")

Merging and combining

orders = pd.DataFrame({"user_id": [1, 2], "total": [100, 250]})
users  = pd.DataFrame({"id": [1, 2], "name": ["Ada", "Grace"]})

pd.merge(orders, users, left_on="user_id", right_on="id", how="left")
# how: "left" | "right" | "inner" | "outer"

pd.concat([df1, df2])              # stack rows
pd.concat([df1, df2], axis=1)        # join columns

Applying functions

df["age"].apply(lambda x: x * 2)          # element-wise on a Series
df.apply(lambda row: row["age"] > 40, axis=1)   # row-wise on a DataFrame
df["name"].map({"Ada": "A", "Grace": "G"})        # value substitution
df["initial"] = df["name"].str[0]                   # vectorized string ops
Prefer vectorized operations: df["age"] * 2 is far faster than df["age"].apply(lambda x: x * 2) because it runs in NumPy's compiled C code instead of a Python-level loop. Reach for .apply() only when the logic can't be expressed as a vectorized expression.
