Math for ML Cheat Sheet

A condensed NumPy/SciPy reference for the linear algebra, probability & statistics, and calculus concepts used throughout machine learning.

Linear Algebra

import numpy as np

v + w      v - w      3 * v          # vector add / sub / scale
np.dot(v, w)   v @ w                    # dot product
np.linalg.norm(v)                         # vector length (L2 norm)

Calculus

# Power rule:        d/dx x^n        = n * x^(n-1)
# Exponential:        d/dx e^x         = e^x
# Logarithm:           d/dx ln(x)        = 1/x
# Sigmoid:              d/dx sigmoid(x)    = sigmoid(x) * (1 - sigmoid(x))
# Chain rule:            d/dx f(g(x))        = f'(g(x)) * g'(x)

def numerical_derivative(f, x, h=1e-6):
    return (f(x + h) - f(x - h)) / (2 * h)

def gradient(f, w, h=1e-6):                  # numerical gradient, any # of params
    grad = np.zeros_like(w)
    for i in range(len(w)):
        wp, wm = w.copy(), w.copy()
        wp[i] += h; wm[i] -= h
        grad[i] = (f(wp) - f(wm)) / (2 * h)
    return grad

# Gradient descent — the core training loop
for step in range(n_steps):
    grad = gradient(loss_fn, w)
    w = w - learning_rate * grad

from scipy import integrate

Import conventions: import numpy as np covers linear algebra and random sampling; from scipy import stats, integrate covers hypothesis testing and integrals. Deep learning frameworks (PyTorch, TensorFlow) automate the gradient computation shown above via autograd.
