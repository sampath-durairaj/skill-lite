About Skill-Lite

Practical tutorials & tools, without the fluff

Skill-Lite is a free collection of hands-on tutorials and browser-based developer tools — built to be fast, clear, and honest about what you actually need to know.

What Skill-Lite is

Skill-Lite started as a simple idea: most technical documentation is either too shallow to be useful or too dense to get through. This site collects the tutorials, cheat sheets, and reference material we wish existed when we were learning Kubernetes, Docker, Spring Boot, Kafka, LangChain, LangGraph, and RAG — written for engineers who want to get productive quickly.

What you'll find here

  • Tutorials — step-by-step guides across Kubernetes, Docker, Spring Boot, Apache Kafka, LangChain, LangGraph, RAG, and Machine Learning.
  • Cheat sheets — quick-reference pages for commands, APIs, and patterns you'll reach for again and again.
  • DevTools — a set of small utilities (SQL formatter, JSON formatter, JWT decoder, cron generator, timezone converter, and more) that run entirely in your browser.
  • Best practices — opinionated guidance drawn from real production experience.

Built to run anywhere

The whole site is pure HTML, CSS, and JavaScript — no build step, no frameworks, no tracking scripts. Every DevTools utility runs locally in your browser; nothing you type into them is ever sent to a server. See our Privacy Policy for details.

Free, and staying that way

There are no accounts, paywalls, or gated content. If Skill-Lite has been useful to you, the best way to support it is to share it with someone who's learning.
