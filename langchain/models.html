Models & LLMs

LangChain wraps every LLM provider behind a unified interface. Switch from OpenAI to Anthropic to a local Ollama model by changing one line — your chain logic stays the same.

Model types

TypeInputOutputUse case
ChatModelList of messagesAIMessageConversation, instruction-following (most common)
LLMPlain stringPlain stringOlder completion-style APIs
EmbeddingsTextFloat vectorSemantic search, RAG, similarity

OpenAI

from langchain_openai import ChatOpenAI, OpenAIEmbeddings

# Chat model
llm = ChatOpenAI(
    model="gpt-4o",           # gpt-4o, gpt-4o-mini, gpt-3.5-turbo
    temperature=0,            # 0 = deterministic, 1 = creative
    max_tokens=1024,
    timeout=30,
    max_retries=2,
)

response = llm.invoke("What is LangChain?")
print(response.content)      # AIMessage.content

# Embeddings
embeddings = OpenAIEmbeddings(model="text-embedding-3-small")
vector = embeddings.embed_query("Hello world")
print(len(vector))           # 1536 dimensions

Local models with Ollama

# Install Ollama and pull a model locally
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3.2
ollama pull nomic-embed-text   # for embeddings
from langchain_community.chat_models import ChatOllama
from langchain_community.embeddings import OllamaEmbeddings

# Completely local — no API key needed
llm = ChatOllama(model="llama3.2", temperature=0)
embeddings = OllamaEmbeddings(model="nomic-embed-text")

response = llm.invoke("What is a neural network?")
print(response.content)

Streaming responses

llm = ChatOpenAI(model="gpt-4o-mini", streaming=True)

# Print tokens as they arrive
for chunk in llm.stream("Write a haiku about Python."):
    print(chunk.content, end="", flush=True)

# Async streaming
import asyncio

async def stream_async():
    async for chunk in llm.astream("Write a poem about Docker."):
        print(chunk.content, end="", flush=True)

asyncio.run(stream_async())

Batch inference

# Process multiple prompts in parallel (respects rate limits)
questions = [
    "What is Docker?",
    "What is Kubernetes?",
    "What is LangChain?",
]

responses = llm.batch(questions)
for q, r in zip(questions, responses):
    print(f"Q: {q}")
    print(f"A: {r.content}\n")

Model configuration & caching

from langchain.globals import set_llm_cache
from langchain.cache import InMemoryCache, SQLiteCache

# Cache responses in memory (useful during development)
set_llm_cache(InMemoryCache())

# Or persist to SQLite (survives restarts)
set_llm_cache(SQLiteCache(database_path=".langchain.db"))

# The same prompt now returns instantly from cache
llm.invoke("What is 2 + 2?")   # hits API
llm.invoke("What is 2 + 2?")   # returns from cache

Token counting & cost estimation

from langchain_openai import ChatOpenAI
from langchain_core.callbacks import UsageMetadataCallbackHandler

callback = UsageMetadataCallbackHandler()
llm = ChatOpenAI(model="gpt-4o-mini", callbacks=[callback])

llm.invoke("Summarise the history of Docker in 3 bullet points.")

print(callback.usage_metadata)
# {'input_tokens': 18, 'output_tokens': 87, 'total_tokens': 105}

Switching providers (the key benefit)

def get_llm(provider: str):
    if provider == "openai":
        from langchain_openai import ChatOpenAI
        return ChatOpenAI(model="gpt-4o-mini", temperature=0)
    elif provider == "anthropic":
        from langchain_anthropic import ChatAnthropic
        return ChatAnthropic(model="claude-haiku-4-5", temperature=0)
    elif provider == "ollama":
        from langchain_community.chat_models import ChatOllama
        return ChatOllama(model="llama3.2", temperature=0)

# Your chain works with any provider
llm = get_llm("anthropic")
response = llm.invoke("What is LangChain?")
Temperature guide: use 0 for factual tasks (code generation, data extraction), 0.3–0.7 for balanced responses, and 0.9–1.0 for creative writing. Keep it at 0 when building RAG or agent pipelines for reproducibility.
