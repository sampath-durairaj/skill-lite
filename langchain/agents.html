Agents & Tools

Agents use an LLM as a reasoning engine to decide which tool to call, when, and what input to pass. The LLM iterates — reason → act → observe → reason again — until it reaches a final answer. This is the ReAct (Reason + Act) pattern.

How an agent thinks (ReAct loop)

1. Thought

  • LLM decides what to do next
  • "I need to search for X"

2. Action

  • LLM picks a tool
  • Provides tool input

3. Observation

  • Tool runs, result returned
  • Back to LLM with result

4. Final Answer

  • LLM has enough info
  • Returns answer to user

Built-in tools

from langchain_community.tools import DuckDuckGoSearchRun, WikipediaQueryRun
from langchain_community.utilities import WikipediaAPIWrapper
from langchain_openai import ChatOpenAI
from langchain.agents import create_tool_calling_agent, AgentExecutor
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# Common built-in tools
search = DuckDuckGoSearchRun()
wiki   = WikipediaQueryRun(api_wrapper=WikipediaAPIWrapper(top_k_results=2))

tools = [search, wiki]

# Agent prompt must include agent_scratchpad
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful research assistant. Use tools to find accurate information."),
    ("human", "{input}"),
    MessagesPlaceholder("agent_scratchpad"),  # ← required for tool call history
])

llm = ChatOpenAI(model="gpt-4o", temperature=0)

agent = create_tool_calling_agent(llm, tools, prompt)
executor = AgentExecutor(agent=agent, tools=tools, verbose=True)

result = executor.invoke({"input": "What is the latest version of LangChain?"})
print(result["output"])

Custom tools with @tool decorator

from langchain_core.tools import tool
import subprocess, json

@tool
def run_shell(command: str) -> str:
    """Run a safe shell command and return the output. Only allow read-only commands."""
    safe_prefixes = ("ls", "cat", "echo", "python --version", "docker ps")
    if not any(command.strip().startswith(p) for p in safe_prefixes):
        return "Error: unsafe command rejected."
    result = subprocess.run(command, shell=True, capture_output=True, text=True, timeout=10)
    return result.stdout or result.stderr

@tool
def get_weather(city: str) -> str:
    """Get current weather for a city (mock)."""
    # In production, call a real weather API here
    return json.dumps({"city": city, "temp": "22°C", "condition": "Sunny"})

@tool
def calculate(expression: str) -> str:
    """Safely evaluate a mathematical expression like '2 * (3 + 4)'."""
    try:
        result = eval(expression, {"__builtins__": {}}, {})
        return str(result)
    except Exception as e:
        return f"Error: {e}"

tools = [run_shell, get_weather, calculate]

Tool with structured input (Pydantic)

from langchain_core.tools import StructuredTool
from pydantic import BaseModel, Field

class SearchInput(BaseModel):
    query: str = Field(description="The search query")
    max_results: int = Field(default=3, description="Number of results to return")

def search_docs(query: str, max_results: int = 3) -> str:
    """Search internal documentation."""
    # Simulate a doc search
    return f"Found {max_results} results for '{query}': [doc1, doc2, doc3]"

search_tool = StructuredTool.from_function(
    func=search_docs,
    name="search_docs",
    description="Search the internal documentation for technical information.",
    args_schema=SearchInput,
)

tools = [search_tool]

Agent with memory

from langchain_core.chat_history import InMemoryChatMessageHistory
from langchain_core.runnables.history import RunnableWithMessageHistory

store = {}
def get_history(sid):
    if sid not in store: store[sid] = InMemoryChatMessageHistory()
    return store[sid]

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful DevOps assistant with access to tools."),
    MessagesPlaceholder("history"),
    ("human", "{input}"),
    MessagesPlaceholder("agent_scratchpad"),
])

agent = create_tool_calling_agent(llm, tools, prompt)
executor = AgentExecutor(agent=agent, tools=tools)

agent_with_memory = RunnableWithMessageHistory(
    executor,
    get_history,
    input_messages_key="input",
    history_messages_key="history",
)

config = {"configurable": {"session_id": "dev_session"}}
agent_with_memory.invoke({"input": "What's the weather in London?"}, config=config)
agent_with_memory.invoke({"input": "And in Paris?"}, config=config)  # remembers context

Agent configuration options

executor = AgentExecutor(
    agent=agent,
    tools=tools,
    verbose=True,           # print thought/action/observation log
    max_iterations=10,      # prevent infinite loops
    max_execution_time=30,  # timeout in seconds
    handle_parsing_errors=True,   # gracefully handle LLM output errors
    return_intermediate_steps=True,  # include tool calls in response
)

result = executor.invoke({"input": "Research the history of Docker"})
# result["output"]              — final answer
# result["intermediate_steps"]  — list of (tool_call, observation) tuples
Safety note: Never give agents unrestricted shell access or file-write permissions in production. Validate and sandbox all tool inputs. Use max_iterations and max_execution_time to prevent runaway loops that rack up API costs.
