Spring Boot Cheat Sheet

A complete reference of Spring Boot annotations, properties, and patterns — organised by category.

Core annotations

AnnotationPurpose
@SpringBootApplicationMain class: enables auto-config, component scan, @Configuration
@ComponentGeneric Spring-managed bean
@ServiceService layer bean
@RepositoryDAO/repository — also translates DB exceptions
@ControllerMVC controller (returns views)
@RestController@Controller + @ResponseBody (returns JSON/XML)
@ConfigurationContains @Bean factory methods
@BeanDeclares a method as a bean factory
@AutowiredInjects a dependency (prefer constructor injection)
@Qualifier("name")Disambiguates when multiple beans of same type exist
@PrimaryMarks default bean when multiple candidates
@Scope("prototype")New instance per injection (default is singleton)
@Profile("dev")Bean or config active only on specified profiles
@LazyDefers bean initialization until first use
@PostConstructCalled after injection, before use
@PreDestroyCalled before context shutdown
@EventListenerMethod handles an application event
@AsyncRun method in a background thread pool
@Scheduled(cron=...)Schedule a method to run periodically
@TransactionalWrap method in a DB transaction
@EnableAsyncActivates @Async support (put on @Configuration)
@EnableSchedulingActivates @Scheduled support

Web / REST annotations

AnnotationPurpose
@RequestMapping("/path")Map URL path to class or method
@GetMapping / @PostMapping / ...HTTP-method-specific shortcuts
@PathVariableExtract value from URL segment
@RequestParamExtract query parameter
@RequestBodyDeserialize request body from JSON
@ResponseBodySerialize return value to JSON
@ResponseStatus(HttpStatus.CREATED)Set HTTP status code for method
@RequestHeaderInject a request header
@CrossOriginEnable CORS for a controller or method
@RestControllerAdviceGlobal exception handler for REST controllers
@ExceptionHandlerHandle a specific exception type
@Valid / @ValidatedTrigger Bean Validation on method argument

Spring Data JPA annotations

AnnotationPurpose
@EntityMarks class as JPA entity (DB table)
@Table(name="...")Override default table name
@IdPrimary key field
@GeneratedValueAuto-generate PK (IDENTITY, SEQUENCE, AUTO)
@ColumnColumn mapping (name, nullable, length, unique)
@OneToMany / @ManyToOneRelationship mapping
@JoinColumnSpecifies FK column for a relationship
@Query("JPQL")Custom JPQL or native SQL on repository method
@ModifyingMarks @Query as an UPDATE/DELETE (must be @Transactional)
@Transactional(readOnly=true)Read-only transaction (performance hint)
@CreationTimestampAuto-set on insert (Hibernate)
@UpdateTimestampAuto-update on every save (Hibernate)

Validation annotations (jakarta.validation)

AnnotationConstraint
@NotNullMust not be null
@NotBlankString must not be null, empty, or whitespace
@NotEmptyCollection/String must not be null or empty
@Size(min, max)String / collection size range
@Min / @MaxNumeric minimum / maximum value
@Positive / @PositiveOrZeroNumber must be positive (or zero)
@EmailMust be a well-formed email address
@Pattern(regexp)Must match regular expression
@Past / @FutureDate must be in the past / future

Testing annotations

AnnotationPurpose
@SpringBootTestFull integration test with Spring context
@WebMvcTest(Ctrl.class)MVC slice — controller + MockMvc only
@DataJpaTestJPA slice — repos + H2 by default
@MockBeanAdd Mockito mock into Spring context
@SpyBeanAdd Mockito spy wrapping real bean
@Mock / @InjectMocksMockito unit test (no Spring context)
@TestJUnit 5 test method
@BeforeEach / @AfterEachRun before/after each test method
@Testcontainers + @ContainerManage Docker containers in tests
@DynamicPropertySourceOverride properties at test time

Useful application.properties / yml keys

server.port=8080
server.servlet.context-path=/api
spring.application.name=my-service
spring.profiles.active=dev

# DataSource
spring.datasource.url=jdbc:postgresql://localhost:5432/mydb
spring.datasource.username=admin
spring.datasource.password=secret

# JPA / Hibernate
spring.jpa.hibernate.ddl-auto=validate
spring.jpa.show-sql=false
spring.jpa.open-in-view=false

# Connection pool (HikariCP)
spring.datasource.hikari.maximum-pool-size=20
spring.datasource.hikari.minimum-idle=5
spring.datasource.hikari.connection-timeout=30000

# Actuator
management.endpoints.web.exposure.include=health,info,metrics,prometheus
management.endpoint.health.show-details=when-authorized

# Logging
logging.level.com.example.myapp=DEBUG
logging.level.org.hibernate.SQL=DEBUG

Maven Wrapper quick commands

./mvnw spring-boot:run                     # run with dev classpath
./mvnw spring-boot:run -Dspring-boot.run.profiles=dev
./mvnw clean package                       # build fat JAR
./mvnw clean package -DskipTests          # skip tests during build
./mvnw test                               # run tests only
./mvnw dependency:tree                    # show dependency tree
./mvnw versions:display-dependency-updates  # check for updates

java -jar target/app.jar
java -jar target/app.jar --spring.profiles.active=prod
java -jar target/app.jar --server.port=9090
